⚡ QUICK TASKS (HOW TO...)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ADD A PROJECT (2 minutes)
1. Click "Projects" tab
2. Click "+ Add New Project"
3. Fill in title, category, description, tech stack
4. Upload image
5. Add live link & GitHub repo
6. Click "Save Project" ✅

WRITE A BLOG POST (5 minutes)
1. Click "Blog" tab
2. Click "+ Write New Article"
3. Add title, category, date, content
4. Upload featured image
5. Click "Publish Article" ✅

ADD A SKILL (1 minute)
1. Click "Skills" tab
2. Click "+ Add Skill"
3. Enter name, drag proficiency slider
4. Click "Add Skill" ✅

CHANGE PASSWORD (2 minutes)
1. Click "Settings" tab
2. Click "Change Password"
3. Enter current: Croesus@2025
4. Create new strong password
5. Confirm & save ✅

BACKUP YOUR DATA (30 seconds)
1. Click "Settings" tab
2. Click "📥 Export Data"
3. Save JSON file to computer ✅

DEPLOY CHANGES (instant)
1. Click "Settings" tab
2. Click "🚀 Sync to Portfolio"
3. Wait 1-2 minutes for Vercel
4. Check live portfolio ✅

Data disappeared?
→ Restore from JSON backup in Settings

Changes not showing?
→ Click "Sync" in Settings → Wait 2 min → Refresh page

Image won't upload?
→ Check size < 5MB, use JPG/PNG, check internet
