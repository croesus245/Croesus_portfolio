## 💡 QUICK FEATURE USAGE

```
🔍 SEARCH (Press "/" key)
   1. Press "/" ← Search bar activates
   2. Type text ← Results show instantly
   3. Click result ← Preview opens
   4. Press Escape ← Close preview

👁️ PREVIEW (Click preview button)
   1. Find any content item
   2. Click 👁️ button ← Modal opens
   3. See formatted preview ← Full appearance
   4. Click "View on Portfolio" ← See live
   5. Click X or press Escape ← Close modal

🚀 SYNC (Click sync button)
   1. Make changes in admin
   2. Click 🚀 button ← Syncing (1-2 seconds)
   3. Watch indicator ← Pulsing animation
   4. Notification appears ← Sync complete
   5. Check portfolio ← Changes live

📊 DASHBOARD (Default view)
   1. Login to admin
   2. See Dashboard tab (active by default)
   3. View all stats ← Real-time data
   4. Check activity ← Last 5 actions
   5. Use quick buttons ← Fast navigation
```
